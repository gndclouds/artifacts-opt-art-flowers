# Opt Art Flowers

A beautiful landing page featuring blooming flowers with optical art effects, built with Next.js and React.

## Features

- Animated blooming flowers with optical art patterns
- Smooth animations and transitions
- Responsive design
- Modern gradient backgrounds
- Interactive canvas-based rendering

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS Modules
